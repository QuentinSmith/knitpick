# Knitpick

A cozy metric knitting calculator. Regauge a pattern to your gauge, size to your
body, and keep your needles and yarns in one place.

> We sweat the small stitches so you don't have to.

This is the web (PWA) build, hosted on GitHub Pages for on-device testing.
The native app is built separately.

## Run locally
Open `index.html` in a browser, or serve the folder over HTTPS
(GitHub Pages, or any static host) to enable install + offline.
